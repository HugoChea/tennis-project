## What you need to start project

* Node
* Docker
* Angular CLI
* Nest CLI

`npm install -g @angular/cli`
`npm install -g @nestjs/cli`


## Run db

`cd docker`
`docker-compose up -d`

## Run backend

`cd api`
`npm install`
`npm start`

## Run frontend

`cd client`
`npm install`
`npm start`

Project should start at : http://localhost:8858/
